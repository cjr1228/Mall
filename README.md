# Mall 移动端商城项目（Uniapp + Servlet）

一个基于 Uniapp + Java Servlet 开发的前后端分离移动端商城系统，包含商品展示、购物车、订单管理、用户登录注册等完整电商核心功能。

## 项目介绍
本项目为移动端电商商城，采用前后端分离架构。
前端使用 Uniapp 开发，支持多端运行；
后端使用 Java Servlet 提供 API 接口，MySQL 数据库存储数据。
项目实现用户登录注册、商品展示、商品详情、购物车、订单管理、个人中心等完整电商功能

## 技术栈
- 前端：Uniapp + Vue
- 后端：Java Servlet
- 数据库：MySQL
- 开发工具：HBuilderX、IDEA
- 数据交互：Axios/Ajax

## 项目结构
├── frontend/  # Uniapp 前端项目
├── dbshopp/   # 数据库文件
├── backend/   # Servlet 后端项目
└── README.md

## 功能模块
- 用户登录、注册
- 商品列表、商品详情
- 购物车添加、删除
- 订单创建、订单列表
- 个人中心
- 后台接口服务

## 运行说明
1. 导入后端项目，配置数据库连接
2. 创建 MySQL 数据库并执行 SQL 文件
3. 启动 Tomcat 运行后端接口
4. 使用 HBuilderX 打开前端项目并运行
